# Setup

**Step 1:**

Download or clone this repo by using the link below:

```
https://github.com/ViktorVoitovich/flutter_notes.git
```

**Step 2:**

Go to project root and execute the following command in console to get the required dependencies: 

```
flutter pub get 
```
